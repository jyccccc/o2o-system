# 点餐系统

## 项目技术

**前端**：用户端Android   +   管理端Web

**后端**：Spring Boot + Mybatis + MySQL

**前后端接口文档**：Swagger2

**API文档**：Postman桌面版

## 项目模块

- **订单模块**：包括提交订单、查询订单、修改订单状态、增加菜品、订单支付（未实现）
- **餐桌模块**：查询餐桌、修改餐桌、增加餐桌（2020-06-19新增）
- **（未实现）公告模块**：查看公告、发布公告
- **菜品模块**：查询所有菜品、查询指定类型的菜品（作为菜单）、增加菜品、删除菜品（未实现）
- **用户模块**：用户登录、用户注册
